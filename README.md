# 🌐 Portfólio Web - Vinicios Weide

## ✨ Sobre o projeto

Este é o meu portfólio pessoal desenvolvido com o objetivo de apresentar meus projetos, habilidades e evolução como desenvolvedor web.

A proposta do site é transformar ideias em experiências digitais modernas, reunindo em um só lugar tudo aquilo que venho aprendendo e construindo ao longo da minha jornada na programação. O portfólio funciona como uma vitrine profissional, permitindo que visitantes conheçam melhor meu trabalho e minhas capacidades técnicas.

## 🚀 Funcionalidades

- 📄 Apresentação pessoal
- 💻 Exibição de projetos desenvolvidos
- 🛠️ Listagem de tecnologias e habilidades
- 📱 Layout responsivo (adaptável a diferentes dispositivos)
- 📬 Formas de contato

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## 🎯 Objetivo

Demonstrar minhas habilidades como desenvolvedor, compartilhar projetos práticos e evoluir constantemente através da criação de interfaces modernas e funcionais.
